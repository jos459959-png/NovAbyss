# Novabyss

A premium aquascaping equipment brand website.

## Description

Novabyss is a high-end minimalist brand showcase website featuring:
- Dark mode design with deep ocean aesthetics
- Smooth scroll animations using Framer Motion
- Horizontal scrolling product carousel with auto-scroll
- Responsive layout for all devices

## Deployment

This project is ready for deployment on Vercel:

1. Push this folder to your GitHub repository
2. Connect the repository to Vercel
3. Deploy with default settings

## Structure

```
├── index.html      # Main entry point
├── assets/         # CSS and JS files
├── images/         # Image assets
└── README.md       # This file
```

## Technologies

- React + TypeScript
- Tailwind CSS
- Framer Motion
- Vite
